## Overview

Jasin transforms Amazon product URLs into clean, customizable product cards that can be embedded on any website. Unlike iframes, our cards are static HTML that loads fast and integrates seamlessly with your site.

## Creating Embed Cards

### Via Dashboard

1. **Paste Amazon URL**: Copy any Amazon product URL
2. **Choose Theme**: Select light, dark, or auto theme
3. **Customize Settings**: Configure affiliate tags and display options
4. **Generate Card**: Get your embed code instantly

### Via API

<CodeGroup>
```bash cURL
curl -X POST https://jasin.app/api/embeds \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "amazon_url": "https://www.amazon.com/dp/B08N5WRWNW",
    "theme": "light",
    "affiliate_tag": "your-tag-20"
  }'
```

```javascript JavaScript
const embed = await fetch('https://jasin.app/api/embeds', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    amazon_url: 'https://www.amazon.com/dp/B08N5WRWNW',
    theme: 'light',
    affiliate_tag: 'your-tag-20'
  })
});
```
</CodeGroup>

## Embed Code Structure

Each embed card consists of two parts:

### 1. HTML Container
```html
<div id="jasin-embed-abc123" class="jasin-embed" data-theme="light">
  <!-- Product card content -->
  <div class="jasin-card">
    <img src="product-image.jpg" alt="Product Name" />
    <div class="jasin-details">
      <h3>Product Title</h3>
      <div class="jasin-price">$29.99</div>
      <a href="amazon-link" class="jasin-button">View on Amazon</a>
    </div>
  </div>
</div>
```

### 2. Tracking Script
```html
<script src="https://jasin.app/embed.js"></script>
```

<Warning>
  Always include the tracking script to enable analytics and ensure proper functionality.
</Warning>

## Card Features

### Product Information
- **Real-time Data**: Fetched via Amazon PAAPI 5.0
- **Product Images**: High-quality images with carousel support
- **Pricing**: Current price with availability status
- **Ratings**: Star ratings and review counts
- **Prime Badge**: Prime shipping indicator

### Customization Options
- **Themes**: Light, dark, and auto (system preference)
- **Colors**: Dynamic color extraction from product images
- **Affiliate Tags**: Custom affiliate tags for Pro users
- **Display Options**: Show/hide ratings, Prime badge, etc.

## Theme System

<Tabs>
  <Tab title="Light Theme">
    Clean, minimal design with light backgrounds perfect for most websites.
    
    ```css
    .jasin-embed[data-theme="light"] {
      background: #ffffff;
      color: #1a1a1a;
      border: 1px solid #e5e5e5;
    }
    ```
  </Tab>
  
  <Tab title="Dark Theme">
    Modern dark design that works great on dark websites.
    
    ```css
    .jasin-embed[data-theme="dark"] {
      background: #1a1a1a;
      color: #ffffff;
      border: 1px solid #333333;
    }
    ```
  </Tab>
  
  <Tab title="Auto Theme">
    Automatically adapts to user's system preference.
    
    ```css
    .jasin-embed[data-theme="auto"] {
      /* Adapts based on prefers-color-scheme */
    }
    ```
  </Tab>
</Tabs>

## Dynamic Color Extraction

Pro users get access to dynamic color extraction that automatically generates theme colors based on the product image:

<AccordionGroup>
  <Accordion title="How it works">
    Our system analyzes the product image and extracts dominant colors to create a cohesive theme that matches the product.
  </Accordion>
  
  <Accordion title="Color Palette">
    - **Primary**: Main brand/product color
    - **Secondary**: Complementary accent color  
    - **Background**: Optimized background color
    - **Text**: High contrast text color
  </Accordion>
  
  <Accordion title="Fallback Colors">
    If color extraction fails, the system falls back to the selected theme's default colors.
  </Accordion>
</AccordionGroup>

## Performance & SEO

### Fast Loading

- **Structured Data**: JSON-LD markup for rich snippets

- **Semantic HTML**: Proper heading hierarchy
- **Mobile Responsive**: Works on all device sizes
