## Overview

Jasin offers three built-in themes to ensure your product embeds look great on any website. Choose the theme that best matches your site's design.

## Available Themes

### Light Theme
Perfect for websites with light backgrounds and traditional designs.

**Features:**
- Clean white background
- Dark text for excellent readability
- Subtle shadows and borders
- Professional appearance
- Works great with most website designs

**Best for:**
- Blogs and content sites
- Business websites
- E-commerce stores
- Traditional web designs

### Dark Theme
Modern dark design that complements dark-themed websites.

**Features:**
- Dark background with light text
- High contrast for readability
- Subtle highlights and accents
- Modern, sleek appearance
- Reduces eye strain in dark environments

**Best for:**
- Dark-themed websites
- Tech blogs and portfolios
- Gaming and entertainment sites
- Modern web applications

### Auto Theme
Automatically adapts to your visitor's system preference for the best user experience.

**Features:**
- Detects user's dark/light mode setting
- Switches themes automatically
- Provides optimal viewing experience
- Respects user preferences
- Future-proof design choice

**Best for:**
- Websites that want to provide the best user experience
- Sites with diverse audiences
- Modern web applications
- Progressive web apps

## How to Choose a Theme

When creating an embed in your dashboard:

1. **Preview all themes** - See how each theme looks with your product
2. **Consider your website** - Choose the theme that best matches your site's design
3. **Think about your audience** - Auto theme provides the best user experience
4. **Test on your site** - Make sure the theme looks good in context

## Setting the Theme

### In the Dashboard
When creating an embed:
1. Paste your Amazon URL
2. Preview the product card
3. Select your preferred theme from the dropdown
4. Click "Create Embed"
5. Copy the generated script tag

### In the Script Tag
The theme is set in the `data-theme` attribute:

```html
<!-- Light theme -->
<script 
  async 
  src="https://getjasin.com/embed.js" 
  data-id="your-embed-id"
  data-theme="light">
</script>

<!-- Dark theme -->
<script 
  async 
  src="https://getjasin.com/embed.js" 
  data-id="your-embed-id"
  data-theme="dark">
</script>

<!-- Auto theme -->
<script 
  async 
  src="https://getjasin.com/embed.js" 
  data-id="your-embed-id"
  data-theme="auto">
</script>
```

## Troubleshooting

<AccordionGroup>
  <Accordion title="Theme not displaying correctly">
    - Verify the data-theme attribute is set correctly
    - Check for CSS conflicts with your website's styles
    - Ensure the script tag is properly formatted
  </Accordion>
  
  <Accordion title="Auto theme not working">
    - Confirm your website supports dark mode detection
    - Check that the user's browser supports prefers-color-scheme
    - Try manually setting light or dark theme as a fallback
  </Accordion>
  
  <Accordion title="Theme doesn't match my site">
    - Consider using CSS to customize the embed container
    - Try a different theme option
    - Contact support for custom styling options (Pro plan)
  </Accordion>
</AccordionGroup>
