## Create Embed

<ParamField body="amazon_url" type="string" required>
  The Amazon product URL to create an embed for
</ParamField>

<ParamField body="theme" type="string" default="light">
  Theme for the embed card. Options: `light`, `dark`, `auto`
</ParamField>

<ParamField body="affiliate_tag" type="string">
  Custom affiliate tag (Pro tier only)
</ParamField>

<ParamField body="show_prime" type="boolean" default="true">
  Whether to show Prime badge
</ParamField>

<ParamField body="show_rating" type="boolean" default="true">
  Whether to show product rating
</ParamField>

### Request Example

<CodeGroup>
```bash cURL
curl -X POST https://jasin.app/api/embeds \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "amazon_url": "https://www.amazon.com/dp/B08N5WRWNW",
    "theme": "light",
    "affiliate_tag": "your-tag-20",
    "show_prime": true,
    "show_rating": true
  }'
```

```javascript JavaScript
const response = await fetch('https://jasin.app/api/embeds', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    amazon_url: 'https://www.amazon.com/dp/B08N5WRWNW',
    theme: 'light',
    affiliate_tag: 'your-tag-20',
    show_prime: true,
    show_rating: true
  })
});

const embed = await response.json();
```
</CodeGroup>

## List Embeds

<Api method="GET" url="https://jasin.app/api/embeds" />

Retrieve a list of your embed cards.

### Query Parameters

<ParamField query="limit" type="number" default="20">
  Number of embeds to return (max 100)
</ParamField>

<ParamField query="offset" type="number" default="0">
  Number of embeds to skip
</ParamField>

<ParamField query="theme" type="string">
  Filter by theme: `light`, `dark`, `auto`
</ParamField>

### Request Example

<CodeGroup>
```bash cURL
curl -X GET "https://jasin.app/api/embeds?limit=10&offset=0" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch('https://jasin.app/api/embeds?limit=10&offset=0', {
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY'
  }
});

const embeds = await response.json();
```
</CodeGroup>
