## Welcome to the Jasin API

The Jasin API provides a simple, RESTful interface for creating and managing Amazon product embed cards. Our API is designed for developers who want to integrate Jasin's functionality into their applications.

## Base URL

All API requests should be made to:

```
https://jasin.app/api
```

## Authentication

All API requests require authentication using an API key in the Authorization header:

```bash
curl -H "Authorization: Bearer YOUR_API_KEY" \
  https://jasin.app/api/embeds
```

## Rate Limits

API requests are rate limited based on your subscription tier:

| Tier | Hourly Limit | Daily Limit |
|------|--------------|-------------|
| Free | 100 requests | 1,000 requests |
| Pro  | 1,000 requests | 10,000 requests |

## Response Format

All API responses are returned in JSON format:

```json
{
  "success": true,
  "data": {
    // Response data
  },
  "meta": {
    "timestamp": "2024-01-15T10:30:00Z",
    "request_id": "req_abc123"
  }
}
```

### Error Responses

Error responses include detailed information:

```json
{
  "success": false,
  "error": {
    "code": "INVALID_AMAZON_URL",
    "message": "The provided Amazon URL is not valid",
    "details": {
      "url": "invalid-url",
      "expected_format": "https://www.amazon.com/dp/ASIN"
    }
  },
  "meta": {
    "timestamp": "2024-01-15T10:30:00Z",
    "request_id": "req_abc123"
  }
}
```

## SDKs and Libraries

### JavaScript/Node.js

```javascript
// Install the Jasin SDK
npm install @jasin/sdk

// Initialize the client
import { JasinClient } from '@jasin/sdk';

const jasin = new JasinClient({
  apiKey: 'your-api-key'
});

// Create an embed
const embed = await jasin.embeds.create({
  amazon_url: 'https://www.amazon.com/dp/B08N5WRWNW',
  theme: 'light'
});
```

### Python

```python
# Install the Jasin SDK
pip install jasin-sdk

# Initialize the client
from jasin import JasinClient

jasin = JasinClient(api_key='your-api-key')

# Create an embed
embed = jasin.embeds.create(
    amazon_url='https://www.amazon.com/dp/B08N5WRWNW',
    theme='light'
)
```

### Example Requests

<CodeGroup>
```bash Create Embed
curl -X POST https://jasin.app/api/embeds \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "amazon_url": "https://www.amazon.com/dp/B08N5WRWNW",
    "theme": "light"
  }'
```

```bash Get Analytics
curl -X GET "https://jasin.app/api/analytics?period=7d" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```bash List Embeds
curl -X GET https://jasin.app/api/embeds \
  -H "Authorization: Bearer YOUR_API_KEY"
```
</CodeGroup>
